About This Project





This is a small weather station that I've created for fun. It uses a set of sensors read by a Raspberry Pi 2 and sits on my desk sampling data every minute, 24 hours a day.

It consists of:

  • Raspberry Pi 2 (Get One Here)
  • AM2302 Temperature / Humidity Sensor
  • BMP180 Temperature / Barometric Pressure Sensor
  • DS18B20 Waterproof Temperature Sensor
  • TSL2561 Digital Lumosity Sensor

The sampling is done by a combination of C++ applications and Python Scripts tied together in a single script which is available on GitHub. It is then sent a .Net WebAPI on Azure (Source soon to be released) and then parsed out on this page.
